# ARQUITECTURA ATM PARA LA GESTIÓN DEL SERVICIO AL CLIENTE - BANCO FIE

Este proyecto implementa un sistema de Cajero Automático (ATM) avanzado, enfocado en la seguridad, inclusión financiera y una interfaz de usuario intuitiva para el Banco FIE.

## 🛠️ Stack Tecnológico
* **Frontend:** SPA con HTML5, CSS3 y jQuery.
* **Backend:** Arquitectura MVC en PHP con API RESTful.
* **Hardware:** Integración con Arduino UNO (RFID, Biometría GT511, GSM).
